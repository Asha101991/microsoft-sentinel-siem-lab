# SOC Dashboard Workbook

## Suggested Panels

- Failed logons by account
- Failed logons by IP address
- Successful logons by host
- Suspicious PowerShell commands
- Top incident severity
- Alerts by MITRE ATT&CK tactic
- Recent high-severity incidents

## Purpose

This workbook helps SOC analysts quickly review authentication trends, suspicious process activity, and open incidents in Microsoft Sentinel.
