# Incident Response Playbook

## Steps

1. Validate the alert.
2. Identify impacted user, host, and IP address.
3. Review related logs in Sentinel.
4. Map activity to MITRE ATT&CK.
5. Contain affected system if necessary.
6. Reset credentials if account compromise is suspected.
7. Document findings in an incident report.
8. Close the incident with classification and notes.
